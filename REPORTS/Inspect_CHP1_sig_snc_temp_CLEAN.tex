\hypertarget{intro}{%
\subsection{Intro}\label{intro}}

Produce yearly plots for \textbf{CHP-1}.

Shows only \textbf{raw data} aspects.

It can use flags to show `CLEAN'/`DIRTY' data.

For `CLEAN' data, it removes from view:

\begin{itemize}
\tightlist
\item
  Bad recordings ranges \texttt{chp1\_bad\_data}
\item
  Tracker async cases \texttt{Async\_tracker}
\item
  Physical recording limits \texttt{chp1\_signal\_lower\_limit()} and \texttt{chp1\_signal\_upper\_limit()}
\end{itemize}

Mark outliers for signal and SD with:

\textbf{mean(variable) -/+ 4 * sd(variable)}

This is just a report it doesn't alter the data.
